### 변수

```
name = '홍길동'
number_of_coin = 3
number_of_coin = '3'
# 두번째와 세번째 변수의 값은 같지만
# 숫자와 문자열 이라는 차이점이 있다.
```

### 출력

```
# 어떠한 문장을 화면에 나타내고 싶을 때
print('Hello World!')
print('나는 3학년 입니다.')
print('나는 '3'학년 입니다.')
# 숫자는 ' ' 안에 넣지 않아도 출력됨
```

### 연산

```
# +, -, *, /, %, **(제곱)

name = '홍' + '길동'
# 이때 name = '홍길동'

year = 2015 + 1991
# 이때  year = 4006

year = '2015' + '1991'
# 이때 year = 20151991
# 문자열의 합과 숫자의 합은 다름!!!

year = '2015' + 1991
# 오류!!! 문자열과 숫자의 합은 오류
```
