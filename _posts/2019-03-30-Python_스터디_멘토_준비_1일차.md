### 변수

```python
name = '홍길동'
number_of_coin = 3
number_of_coin = '3'
# 두번째와 세번째 변수의 값은 같지만
# 숫자와 문자열 이라는 차이점이 있다.
```

### 출력

```python
# 어떠한 문장을 화면에 나타내고 싶을 때
print('Hello World!')
print('나는 3학년 입니다.')
print('나는 '3'학년 입니다.')
# 숫자는 ' ' 안에 넣지 않아도 출력됨
```

### 연산

```python
# +, -, *, /, %, **(제곱)

name = '홍' + '길동'
print(name)
>>홍길동
# 문자열끼리의 합은 문자열연결

print(name*3)
>>홍홍홍
# 문자열에 숫자를 곱하면 반복연산

year = 2015 + 1991
# 이때  year = 4006

year = '2015' + '1991'
# 이때 year = 20151991
# 문자열의 합과 숫자의 합은 다름!!!

year = '2015' + 1991
# 오류!!! 문자열과 숫자의 합은 오류

num=2
print(num**5)
>>32
# ** 연산은 제곱으로 **뒤에 나오는 숫자만큼 제곱한다
```
